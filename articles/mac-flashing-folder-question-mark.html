How to Fix a Mac Flashing Folder With a Question Mark

A flashing folder with a question mark on a Mac usually means the computer cannot find its startup disk. This can happen if the hard drive fails, the drive cable is damaged, or macOS is not installed.

Common Causes

  • Failed hard drive or SSD
  • Loose or damaged drive cable
  • Corrupted macOS installation

Steps to Try

Restart the Mac while holding the Option key to check available startup disks. If no disk appears, the computer may not be detecting the internal drive.
